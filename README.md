## 📌 Project Overview

This project presents an interactive **Annual Sales Analysis Dashboard** for **Vrinda Store** using **Microsoft Excel**. The objective is to analyze sales performance, customer demographics, order status, sales channels, and regional performance to derive meaningful business insights.

The dashboard enables users to filter data dynamically and explore sales trends for better business decision-making.

---

## 🎯 Objectives

- Analyze annual sales performance.
- Compare sales and order trends.
- Identify top-performing states.
- Analyze customer demographics.
- Compare sales between men and women.
- Evaluate different sales channels.
- Study order status distribution.
